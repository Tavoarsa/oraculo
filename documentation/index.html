App Features

Restab is Restaurant tablewise billing system(POS). complete solution for Restaurant. This CMS Includes almost everything needed in a Restaurant billing system. We made this fully responsive so that every device can get an access.We use CodeIgniter as a framework.its very easy to changes. This Restab Management System allows multiple admin and multiple User. admin can observe whatever a User doing what? How many sells posts by a Waiter? etc. Cashier(Waiter) can only sell products and does not have any access to delete anything form the system. we provide user-right also for any other user. admin set and reset password or permission of access.

Base requirment

  • Application is developed in  CodeIgnitor 3.x version. For this you must required Linux hosting.  find help regarding codeigniter  :  https://www.codeigniter.com/user_guide      
  • For database required MySql.

 

 

Installation of Application

- Upload

  • Upload "Main" folder to your server with your home directory or in subdomain directory.

- Create Database

  • Create Database and assign User permission to database.  If you don't wont to follow installation process direct can import default database.
  • "DATABASE" folder in package files contain database. you can directly import this file to get sample data and then edit all information in Main 
  • database Login details : 

User : admin

Pass : admin

 

- Configure Database

                Change your database credentials to  "application/database.php" file. change following lines..

$db['default'] = array(

                'dsn'       => '',

                'hostname' => 'localhost',

                'username' => 'root',

                'password' => '',

                'database' => 'db_name',

 

 

- Config Site URL

                Change site url in following configuration file "application/config.php".  Update Base URL with your URL.

 

 

 

 

How To Use

  • Open your base url if you not directly import sample data then you must follow installation process.

  • process...finish.  if some time it give you error then refresh page again.

 

                1. Login Information Default user = admin Password = admin

                                This information will allow Restab to login with this details.

                2. Restab Configure Information

                                All information about Your Restaurant You can add your store name logo and other tax detail sms Api with backup your database config.

                3. Create Add Category.

                                Settings for Product you need to Create Category first, Go to Category and add it.

                4. Create Add Product Option.

                                Settings for Product Option you need to Create Option, Go to Product Option and add it.

                5. Create Add Product

                                Settings for Product you need to Create product, Go to Product and add it.
